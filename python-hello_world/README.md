# Python - Hello, World

First ALU Python project. Practice the Python interpreter, `print`, strings, integers and floats.

All Python files use `#!/usr/bin/python3` and must end with a new line. Style is checked with pycodestyle 2.7.*.

## Tasks

0. **0. Run Python file** — `0-run`
1. **1. Run inline** — `1-run_inline`
2. **2. Hello, print** — `2-print.py`
3. **3. Print integer** — `3-print_number.py`
4. **4. Print float** — `4-print_float.py`
5. **5. Print string** — `5-print_string.py`
6. **6. Play with strings** — `6-concat.py`
7. **7. Copy - Cut - Paste** — `7-edges.py`
8. **8. Create a new sentence** — `8-concat_edges.py`
9. **9. Easter Egg** — `9-easter_egg.py`
